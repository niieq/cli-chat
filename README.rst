CLI Chat
============================

A commandline Chat Application

Installation
------------

.. code-block:: bash

    $ vagrant up
    $ vagrant ssh

Usage
-----

.. code-block:: python
	
	# Show client's messages - ./client.py username no. of msgs
	./client.py nii 10

	# Show all clients
	./client.py users

	# Chat with user
	./client.py
	
	# $ Welcome! Enter your username > nai
	# $ nii: Hello nii
