MPower Python Client Library
============================

This is a Python library for accessing the MPower Payments HTTP API.

Installation
------------

.. code-block:: bash

    $ vagrant up
    $ vagrant ssh

Usage
-----

.. code-block:: python
	
	# Show client's messages - ./client.py username no. of msgs
	./client.py nii 10

	# Show all clients
	./client.py users

	# Chat with user
	./client.py

	.. code-block:: bash

		$ Welcome! Enter your username > nai
		$ nii: Hello nii
