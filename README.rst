CLI Chat
============================

A commandline Chat Application

Installation
------------

.. code-block:: bash

    $ vagrant up
    $ vagrant ssh

Usage
-----

.. code-block:: bash
	
	# Show client's messages - ./client.py username no. of msgs
	$ ./client.py nii 10

	# Show all clients
	$ ./client.py users

	# Chat with user
	$ ./client.py

	# Enter username to register or login
	$ Welcome! Enter your username > nai

	# To send a user a message, type username: message
	$ nii: It works!
